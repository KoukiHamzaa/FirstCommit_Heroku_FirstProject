Display settings - top

  1. Feed name - Name of the feed for the refrence so that you can know which feed you are using.
  2. Feed type - You can select the feed type here. You can select which feed you want to display - your feeds, recent media, your likes, popular feeds, any user, tag.
  3. Image size - You can select the option which image size you want to use. Available image sizes - thumbnail, medium and large.
  4. Show instagram link - Option to show link to instagram. If the content type is video video icon will appear and if content type is image camera icon will appear.
  5. show user image - Option to show the user image.
  6. Show image like count - Option to show image like count.
  7. Show image comment count - Option to show image comment count.
  8. Counter display format - Here you can choose options for display of the counter formats.
  9. Lightbox options - Here you can enable/disable the lightbox options. And if lightbox enabled you can select which lightbox you want to display.
  10. No of images to display - Here you can chooose the number of images you want to display on page load. Set the number of images according to the layout.
  11. Theme accent color - You can set the hover background color here. You can set the opacity of color as well. If not selected default color will be used.
  12. Hover text color - You can set the hover text colors here. If nothing selected default color will be used.
    13. Now when you click on "Add" button a shortcode will be generated with the specific id for that feed

Use the function to display within template or theme files.

The created feeds will be available in the widets area as well.
