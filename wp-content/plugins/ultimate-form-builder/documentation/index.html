New Form

You can click on "New Form" sub menu to create a new form.You can select the single or multi step form as per your requirement. Once you create a new form, you will be redirected to the main Form Builder page where you can add various fields to build the form.

Form Builder

Currently, available fields are :

HTML Elements
  • Text Field
  • Textarea
  • Email
  • Dropdown
  • Checkbox
  • Password
  • Hidden Field
  • Number
  • Submit & Reset Button
  • Captcha
UI Elements
  • Datepicker
  • Datepicker Date Range
  • Dropdown Date
  • Dropdown Date Range
Custom Elements
  • File Uploader
  • Custom Texts
  • Agreement Block
  • URL Field
  • WYSIWYG
Survey Elements
  • Star Ratings
  • Like Dislike
  • Choice Matrix

Note: Please use only one Captcha Field in the form.Using multiple Captcha may create conflict in the form and may not work properly.

Display Settings

In this tab you can configure all the necessary display and layout setting for the form.You can choose the pre designed templates for the form.Altogether there are 4 form templates including Default Template.

You can configure the Form Width in this section.Please provide the form width in % for the Mobile Device Compatibility.

You can also disable all the CSS styles provided the plugin so that you can custom design by simply checking "Disable Plugin Styles".

Email Settings

You can configure all the email related settings from this section.You can add as many receiver's email address as you want from this section.You can also configure From Email, From Name, Email Subject & Message Format.

You can also configure email auto respond in this section through which you can configure the auto response email to any of the email field available in the form.

Conditional Logic

You can configure all the conditional logics in this section. You can configure three types of conditional logics.

  • Display Logic - Can be used to display and hide certain form elements as per the value of some other form elements
  • Redirect Logic - Can be used to redirect the user to specific pages as per the value of some form elements
  • Email Logic - Can be used to email specific email addresses as per the value of some form elements

Form Entries

You can view all the form entries in the Form Entries section.You can filter the form entries as per the Forms and also export to CSV after the filter.

Form Export/ Import

You can export and import the forms from this section. Please note that you can export the form entries from form entries section.

Shortcode

As soon as you create a form you will be able to view the shortcode in the Form Builder page which you can paste in post or page content area to display the form in the frontend.
